# Project conventions for Claude

## Commit messages and pull requests

Do not include co-author trailers or session/tool attribution in commit
messages or pull request bodies. Specifically:

- No `Co-Authored-By:` trailers (including `Co-Authored-By: Claude ...`).
- No `Claude-Session:`, `Generated with Claude Code`, or similar tool
  attribution lines.
- No `🤖 Generated with ...` footers.

Write the commit or PR body as if a human maintainer authored it — the
subject/body should describe the change and the reasoning, nothing else.
